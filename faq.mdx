## Troubleshooting Common Issues

Follow these steps to resolve frequent problems like slow loading or sync errors.

<Steps>
  <Step title="Check Network Connection" icon="wifi">
    Verify your internet speed exceeds `>5Mbps`. Restart your router if needed.
  </Step>
  <Step title="Clear Cache" icon="refresh-cw">
    ```bash
    # Clear browser cache
    rm -rf ~/.cache/documentation-platform
    ```
    Refresh the page with <kbd>Ctrl</kbd>+<kbd>F5</kbd>.
  </Step>
  <Step title="Restart Services" icon="refresh-ccw">
    Stop and restart the documentation server:
    ```bash
    npm run stop
    npm run start
    ```
  </Step>
  <Step title="Check Logs" icon="file-text">
    Review console logs for errors like `ECONNREFUSED`.
  </Step>
</Steps>

## Best Practices for Organization

Organize your docs effectively across platforms.

<Tabs>
  <Tab title="GitHub" icon="github">
    Use folders for hierarchy:
    ```
    docs/
    ├── guides/
    │   └── quickstart.mdx
    ├── api/
    │   └── endpoints.mdx
    └── faq.mdx
    ```
    Commit changes with descriptive messages.
  </Tab>
  <Tab title="Local Development" icon="code">
    Maintain a `sidebar.json` for navigation:
    ```json
    {
      "docs": [
        { "type": "doc", "id": "faq" },
        { "type": "category", "label": "Guides", "items": ["quickstart"] }
      ]
    }
    ```
  </Tab>
  <Tab title="Team Collaboration" icon="users">
    Assign roles in your repo settings. Review changes via pull requests before merging.
  </Tab>
</Tabs>

<ExpandableGroup>
  <Expandable title="How do I recover a deleted page?" default-open="true">
    Navigate to your repo history on GitHub. Use `git revert <commit-hash>` to restore:
    ```bash
    git revert abc1234
    git push origin main
    ```
    Pages rebuild automatically within 2 minutes.
  </Expandable>
  <Expandable title="Why is my search not working?">
    Ensure indexing completes after updates. Wait `>5 minutes` or trigger manual rebuild via dashboard.
  </Expandable>
  <Expandable title="How to customize the theme?">
    Edit `tailwind.config.js`:
    ```javascript
    module.exports = {
      theme: {
        extend: {
          colors: {
            brand: '#3B82F6'
          }
        }
      }
    };
    ```
    Restart the dev server.
  </Expandable>
</ExpandableGroup>

## Integration Tips

Embed your docs or connect via API.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch latest docs
  const response = await fetch('https://docs.abdelrahmanyousef.com/api/pages', {
    headers: { 'Authorization': `Bearer ${API_TOKEN}` }
  });
  const pages = await response.json();
  ```
  ```python
  # Fetch latest docs
  import requests
  response = requests.get(
      'https://docs.abdelrahmanyousef.com/api/pages',
      headers={'Authorization': f'Bearer {API_TOKEN}'}
  )
  pages = response.json()
  ```
</CodeGroup>
