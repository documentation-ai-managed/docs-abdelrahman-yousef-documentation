## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement", "bugfix"]}>

## New Features

- Added AI-powered search with semantic matching for faster documentation discovery.
- Introduced customizable themes using your brand color `#3B82F6`.
- Enabled real-time collaboration for teams editing docs simultaneously.

## Improvements

- Enhanced MDX parsing speed by 40% for larger documentation sites.
- Improved mobile responsiveness with better touch targets and navigation.

## Bug Fixes

- Fixed mismatched JSX tag parsing errors in component validation.
- Resolved nested code block rendering issues with 4-backtick fences.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Mermaid diagram support using secure code block syntax.
- Added `Update` component for structured changelog entries.

## Bug Fixes

- Corrected special character escaping for `{`, `}`, `<`, `>` in prose.
- Fixed frontmatter YAML parsing by enforcing no blank lines.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched core MDX components: `Callout`, `Card`, `Steps`, `Tabs`, and more.
- Built-in support for 5-8 diverse components per documentation page.

## Breaking Changes

- Removed legacy H1 headings; always start with H2 for frontmatter titles.

</Update>
